# Wallet Setup Guide

Complete setup for Apple Wallet and Google (Android) Wallet.

---

## Quick Reference

| | Apple Wallet | Google Wallet |
|---|---|---|
| Cost | $99/yr (Apple Developer) | Free |
| Credential | 3 PEM cert files | 1 JSON key file |
| Output | `.pkpass` file | JWT URL |
| Works on | iPhone (iOS 6+), Apple Watch | Android (Google Wallet app) |
| Time to set up | ~45 min | ~15 min |

---

## Google Wallet (Android) — Do This First (Free)

### Step 1 — Google Cloud project
1. Go to [console.cloud.google.com](https://console.cloud.google.com)
2. Click the project dropdown → **New Project**
   - Name: `makersco-wallet`
3. **APIs & Services → Library** → search **"Google Wallet API"** → Enable

### Step 2 — Service account key
1. **IAM & Admin → Service Accounts → Create Service Account**
   - Name: `wallet-signer`
   - No role needed at creation
2. Click the created service account → **Keys tab → Add Key → JSON**
3. Download the JSON file
4. Save it as: `certs/google-service-account.json` ← already gitignored

### Step 3 — Enroll as Wallet issuer
1. Go to [pay.google.com/business/console](https://pay.google.com/business/console)
2. Sign in with the **same Google account** as your Cloud project
3. Complete enrollment: business name + contact email
4. Copy your **Issuer ID** (19-digit number)

### Step 4 — Grant Wallet permission to service account
1. Back in Google Cloud Console → **IAM & Admin → IAM**
2. Click **Grant Access**
   - New principal: paste the service account email (from the JSON key file, `client_email` field)
   - Role: search **"Google Wallet Object Issuer"** → Select → Save

### Step 5 — Generate the pass
```bash
# Update your Issuer ID first:
#   Open src/utils/google-wallet-generator.js
#   Set CONFIG.issuerId to your 19-digit Issuer ID (or use env var)

export GOOGLE_WALLET_ISSUER_ID="1234567890123456789"
npm run generate:google
```

Output: a URL like `https://pay.google.com/gp/v/save/eyJ...` — paste this into your card's "Add to Google Wallet" button.

## Apple Wallet (iOS) — Requires Apple Developer Account

### Prerequisites
- Apple Developer account: $99/yr at [developer.apple.com](https://developer.apple.com)
- A Mac (Keychain Access is Mac-only)

### Step 1 — Create a Pass Type ID
1. Go to [developer.apple.com](https://developer.apple.com) → **Certificates, Identifiers & Profiles**
2. **Identifiers → + → Pass Type IDs**
   - Description: `MakersCo Business Card`
   - Identifier: `pass.co.makersco.soonfook`
3. Register it

### Step 2 — Generate and download the certificate
1. On the Pass Type ID page → **Edit → Create Certificate**
2. Follow the CSR instructions (Keychain Access → Certificate Assistant → Request a Certificate from CA)
3. Upload the `.certSigningRequest` file
4. Download the resulting `.cer` file
5. Double-click it — it installs into Keychain Access

### Step 3 — Export certs to PEM
Open Terminal and run these 4 commands:

```bash
# 1. Export your Pass cert + key from Keychain as .p12
#    Keychain Access → My Certificates → right-click "Pass Type ID: pass.co.makersco.soonfook" → Export
#    Save as: cert.p12  (set an export password if prompted)

# 2. Extract the certificate (public)
openssl pkcs12 -in cert.p12 -clcerts -nokeys -out certs/signerCert.pem -legacy

# 3. Extract the private key
openssl pkcs12 -in cert.p12 -nocerts -nodes -out certs/signerKey.pem -legacy
# If you set an export password, add: -passin pass:YOUR_PASSWORD

# 4. Download Apple WWDR G4 root cert (free, public)
curl -o certs/wwdr.der https://www.apple.com/certificateauthority/AppleWWDRCAG4.cer
openssl x509 -in certs/wwdr.der -inform DER -out certs/wwdr.pem
rm certs/wwdr.der
```

### Step 4 — Add your Team ID
1. Go to [developer.apple.com](https://developer.apple.com) → top-right shows your Team ID (10 chars)
2. Set it:
```bash
export PASS_TYPE_ID="pass.co.makersco.soonfook"
export PASS_TEAM_ID="ABCD123456"
```

### Step 5 — Generate pass icons (if not done yet)
```bash
npm run generate:icons
# Generates placeholder icons in src/assets/pass-icons/
# Replace with real brand images before production
```

### Step 6 — Generate the .pkpass
```bash
npm run generate:apple
# Output: output/card.pkpass
```

### Step 7 — Host the .pkpass file
Upload `output/card.pkpass` to your hosting (e.g. `makersco-card-live` GitHub Pages repo).

Then update the "Add to Apple Wallet" button in your card HTML:
```html
<a href="https://lousycoder96.github.io/makersco-card-live/card.pkpass">
  <img src="https://developer.apple.com/wallet/add-to-apple-wallet-badge.svg"
       alt="Add to Apple Wallet" height="44">
</a>
```
